# Aeon Village
Emotional ARPG

## Prerequisites

- Node 10
- yarn
- A recent browser (Chrome 67)
- [Tiled](https://www.mapeditor.org/) for map edition

## Installation

```bash
yarn       # install the dependencies
yarn start # run the dev server
```
